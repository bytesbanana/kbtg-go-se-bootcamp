#Lecture Note

### Restore file before commit (Discard changes)
```bash
git restore [file]
```

### Unstaged the filed
```bash
git restore  --staged [file]
```

### Revert commit (by create new commit that reverse changes)
 ```bash
git revert [commit]
 ```
